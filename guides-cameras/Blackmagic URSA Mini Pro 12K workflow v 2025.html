Введение: зачем workflow 12K и кому он нужен?

Blackmagic URSA Mini Pro 12K - это революция в цифровом кино: Super 35 сенсор 12 288 x 6480 (80 МП), 14 ступеней динамики, Blackmagic RAW, 12K@60p, 8K@120p, 4K@240p, Generation 5 Color Science и интеграция с DaVinci Resolve Studio. Такой workflow нужен для полнометражных фильмов, рекламы, VFX, IMAX, клипов, архивной съёмки и проектов с максимальным запасом на будущее.

Съёмка: оптимальные настройки URSA Mini Pro 12K

  • Формат: Blackmagic RAW 12 бит (Q0/Q3/5:1/8:1/12:1) - выбирайте компрессию под задачу. Q0 - максимум качества, 8:1 - баланс для длительных съёмок.
  • Разрешение: 12K@60p, 8K@120p, 4K@240p (Super16 crop). Для VFX и IMAX - только 12K.
  • ND-фильтры: Встроенные 2/4/6 ступеней, IR-балансировка - важно для чистоты цвета на ярком свету.
  • Оптика: PL/EF/Nikon F, cine-объективы, анасморфот, vintage - раскрывают потенциал сенсора.
  • Запись: Dual CFast/UHS-II SD/USB-C SSD/NVMe (URSA Mini Recorder) - для 12K Q0 нужен NVMe SSD.
  • Метаданные: Записывайте slate, lens data, LUT, отметки для VFX и цветокора.

Пошаговая инструкция: съёмка и бэкап 12K RAW

  1. Подготовьте SSD/NVMe и карты CFast/SD: отформатируйте в камере, убедитесь в скорости записи (не менее 900 МБ/с для 12K Q0).
  2. Настройте Blackmagic RAW: выберите компрессию (Q0/Q3/5:1/8:1), битрейт, LUT, метаданные.
  3. Снимайте с запасом по экспозиции: 14 стопов динамики позволяют тянуть детали в тенях и светах.
  4. Сразу после съёмки делайте три копии материала: основной SSD, резервный HDD, облако или NAS.
  5. Проверяйте целостность файлов (MD5/XXHash): используйте Hedge, ShotPut Pro, DaVinci Resolve Media Management.

Монтаж: работа с 12K RAW в DaVinci Resolve Studio

  • DaVinci Resolve Studio (в комплекте) открывает Blackmagic RAW 12K напрямую, без прокси и перекодирования.
  • Blackmagic RAW SDK оптимизирован под CPU/GPU (Metal, CUDA, OpenCL) - даже на ноутбуках 12K воспроизводится в реальном времени.
  • Монтаж возможен в 12K, 8K, 4K, с динамическим изменением таймлайна - удобно для VFX, IMAX, соцсетей.
  • Прокси не нужны: BRAW позволяет работать с оригиналами, снижая нагрузку за счёт компрессии.

Цветокоррекция: Generation 5 Color Science и HDR

  • Generation 5 Color Science: новая кривая, точные цвета, плавные скинтоны, идеальна для VFX и кино.
  • Работа с BRAW в Resolve - доступен полный контроль: ISO, баланс, экспозиция, tint, saturation, highlight recovery, LUT.
  • HDR-грейдинг: поддержка Rec.2020, Dolby Vision, PQ/HLG, экспорт в HDR10+ для YouTube/Netflix.
  • Возможность делать “look development” для VFX, экспорт LUT для onset-мониторов.

VFX и композитинг: преимущества 12K RAW

  • Съёмка в 12K - идеальна для keying, трекинга, rotoscoping, VFX-проекций, 3D-рендеринга.
  • Экспорт в EXR/TIFF 16 бит для Nuke, Fusion, After Effects - максимум деталей для композитинга.
  • Возможность делать “виртуальный multicam” из одного 12K кадра - кропы, панорамирование, стабилизация без потери качества.
  • Oversampling: 12K > 8K/4K даёт сверхчистую картинку без муара и алиасинга.

Пошаговая инструкция: постпродакшн в DaVinci Resolve Studio

  1. Импортируйте BRAW в Media Pool: Resolve читает метаданные, LUT, lens data, маркеры.
  2. Монтаж на Cut/Edit Page: Работайте с нативными 12K, используйте timeline scaling для ускорения.
  3. Цветокоррекция на Color Page: Регулируйте ISO, экспозицию, tint, saturation, применяйте LUT, работайте с HDR.
  4. VFX на Fusion Page: Keying, rotoscoping, трекинг, композитинг - экспорт в EXR/TIFF для внешних VFX.
  5. Аудио на Fairlight Page: Запись, монтаж, микс, мастеринг.
  6. Экспорт на Deliver Page: 12K/8K/4K ProRes, H.265, DCP, YouTube, Netflix, IMAX.

Лайфхаки и советы по 12K workflow

?? Практические советы

Для экономии места используйте компрессию 8:1 или 12:1 - для большинства задач разницы с Q0 не видно.

Снимайте с запасом по экспозиции: BRAW тянется лучше, чем ProRes, но не “пережигайте” хайлайты.

Используйте SSD/NVMe с охлаждением - перегрев может привести к потере данных.

Работайте с оригиналами в Resolve - прокси нужны только для слабых ноутбуков.

Для VFX экспортируйте EXR 16 бит из Resolve - это максимум для keying и композитинга.

FAQ – часто задаваемые вопросы о workflow URSA Mini Pro 12K

  • Можно ли монтировать 12K RAW на ноутбуке?
    Да, BRAW оптимизирован для CPU/GPU, даже MacBook Pro M2/M3 справляется с 12K timeline.
  • Нужны ли прокси для 12K?
    Нет, в большинстве случаев BRAW позволяет работать с оригиналами, прокси - только для слабых ПК.
  • Какой диск использовать для записи 12K?
    Только NVMe SSD (URSA Mini Recorder), скорость не ниже 900 МБ/с для Q0.
  • Можно ли снимать 8K/4K без кропа?
    Да, благодаря in-sensor scaling 8K/4K RAW пишется с полного сенсора без потери угла обзора.
  • Какая компрессия BRAW оптимальна?
    Для кино и VFX - Q0/Q3, для рекламы/YouTube - 5:1/8:1, для длительных съёмок - 12:1.
