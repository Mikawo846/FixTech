Введение: почему X-H2S - выбор для 6K-видео?

Fujifilm X-H2S - одна из самых технологичных кроп-камер для видео в 2025 году. Она поддерживает съёмку в 6.2K (6240?4160) до 30p, 4K до 120p, ProRes и H.265, выдаёт широкий динамический диапазон (F-Log2), эффективную стабилизацию и автофокус с трекингом лиц и глаз. Это универсальный инструмент для рекламы, документалистики, YouTube, кино и коммерческих проектов.

  • 6.2K (3:2) - максимальная детализация, гибкость для кропа и постобработки
  • Внутренняя запись ProRes 422 HQ - профессиональное качество и быстрый монтаж
  • F-Log2 - широкий динамический диапазон и гибкая цветокоррекция
  • Высокоскоростной автофокус и эффективная IBIS
  • Длительная запись без перегрева (до 240 мин с вентилятором)

Пошаговый алгоритм настройки видео 6K на Fujifilm X-H2S

  1. В меню “Movie Mode” выберите 6.2K 30p (3:2 или 16:9) и нужный кодек (ProRes или H.265).
  2. В разделе “Movie Setting” активируйте профиль F-Log2 для максимального динамического диапазона.
  3. Установите ISO 640 (базовое для F-Log2), баланс белого вручную под свет.
  4. Включите автофокус с распознаванием лиц/глаз (Face/Eye AF) или ручной фокус для креативных задач.
  5. Проверьте уровень аудио, подключите внешний микрофон или рекордер.
  6. Включите стабилизацию IBIS (и OIS - если есть в объективе) для съёмки с рук.
  7. Проверьте свободное место на CFexpress-карте, заряд батареи, наличие вентилятора для долгих сессий.
  8. Для цветокоррекции в посте - снимайте в F-Log2, экспонируйте с запасом вправо (ETTR).

Лайфхаки и советы для идеального 6K-видео на X-H2S

?? Практические советы

Используйте вентилятор Fujifilm FAN-001 для долгих сессий - он предотвращает перегрев даже летом.

Снимайте в F-Log2 и экспонируйте “вправо” - это даст максимум информации для цветокоррекции.

Для быстрой публикации снимайте в H.265 и используйте пленочные симуляции прямо в камере.

Проверяйте уровень аудио и используйте наушники для мониторинга - встроенный микрофон только для “черновика”.

Для динамики используйте “Wide/Tracking AF” и следящий автофокус по глазам.

FAQ – часто задаваемые вопросы о видео 6K на X-H2S

  • Можно ли снимать 6K дольше 30 минут?
    Да, X-H2S поддерживает длительную запись - до 240 минут с вентилятором.
  • Какой кодек выбрать для монтажа?
    ProRes 422 HQ - для профи и монтажа без перекодирования, H.265 - для экономии места и быстрой публикации.
  • Нужна ли быстрая карта памяти?
    Да, для ProRes HQ - только CFexpress Type B (VPG400+), для H.265 - можно SD UHS-II.
  • Можно ли снимать в F-Log2 и сразу получить “киношную” картинку?
    Нет, F-Log2 требует цветокоррекции. Для “готового” цвета используйте пленочные симуляции.
  • Работает ли автофокус по глазам в 6K?
    Да, X-H2S поддерживает трекинг лиц и глаз даже в 6K.
