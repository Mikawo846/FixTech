# Кодировка проекта

## Текущая кодировка
Вся кодировка проекта установлена на **UTF-8**. Все HTML, CSS, JS и другие текстовые файлы должны быть сохранены в UTF-8.

## Скрипты конвертации кодировки
Следующие Python-скрипты выполняют массовую обработку файлов и могут испортить кодировку:
- `convert_html_to_utf8.py` - конвертирует все HTML файлы в UTF-8
- `convert_back_to_cp1251.py` - конвертирует HTML файлы обратно в cp1251
- `fix_cyrillic_encoding.py` - исправляет кодировку кириллицы в guides-laptops
- `fix_cyrillic_encoding_v2.py` - расширенное исправление кодировки кириллицы
- `fix_double_utf8.py` - исправляет двойную UTF-8 кодировку
- `restore_cyrillic.py` - восстанавливает кириллицу из бэкапов или исправляет кодировку

### Важно!
Эти скрипты **отключены по умолчанию** и при запуске выводят предупреждение без выполнения действий.

**Запускать только вручную и только на копии проекта!**

Чтобы запустить скрипт:
1. Сделайте копию всего проекта
2. В скрипте раскомментируйте строку `# main()` в блоке `if __name__ == "__main__"`
3. Запустите скрипт на копии
4. Проверьте результат и примените к основному проекту только если всё правильно

## Настройки проекта
- `.vscode/settings.json` настроен на UTF-8
- Нет автоматических задач, хуков или пайплайнов, запускающих эти скрипты
- Все изменения кодировки должны выполняться вручную с осторожностью
