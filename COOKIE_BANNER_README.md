# Cookie Баннер - Инструкция по использованию

## Обновление от 5 февраля 2026

### Что было изменено:

1. **Дизайн баннера**: Изменен с фиксированного внизу страницы на всплывающее уведомление
2. **Поведение**: Теперь баннер появляется на КАЖДОЙ странице при загрузке
3. **Анимация**: Добавлена плавная анимация появления и исчезновения
4. **Стили**: Современный градиентный дизайн с улучшенной адаптивностью

### Основные файлы:

- `cookie-consent.js` - Логика работы баннера
- `styles.css` - Стили (строки 1620-1770)
- `test-cookie.html` - Тестовая страница для проверки

### Как это работает:

1. При загрузке любой страницы через 1 секунду появляется баннер
2. Баннер всплывает снизу по центру с анимацией
3. При нажатии "Принять" баннер скрывается с анимацией
4. Аналитика загружается только после принятия

### Стили баннера:

- **Позиция**: `position: fixed`, `bottom: 20px`, `left: 50%`, `transform: translateX(-50%)`
- **Размер**: `max-width: 90%`, `width: 600px`
- **Дизайн**: Градиентный фон `linear-gradient(135deg, #1e3c72 0%, #2a5298 100%)`
- **Анимация**: Появление снизу с cubic-bezier переходом
- **Адаптивность**: Оптимизирован для мобильных устройств

### Классы:

- `.cookie-banner` - Основной контейнер
- `.cookie-banner.hidden` - Скрытое состояние
- `.cookie-banner.show` - Активное состояние с анимацией
- `.cookie-banner__text` - Текст уведомления
- `.cookie-banner__buttons` - Контейнер кнопок
- `.cookie-btn--primary` - Основная кнопка "Принять"

### Проверка работы:

1. Откройте любую страницу сайта
2. Подождите 1 секунду - должен появиться баннер
3. Нажмите "Принять" - баннер должен скрыться с анимацией
4. Обновите страницу - баннер должен появиться снова

### Совместимость:

- Все основные страницы сайта обновлены
- Статьи и гайды содержат баннер
- Отчеты сравнения также включены
- Тестовая страница: `test-cookie.html`

### Примечания:

- Баннер появляется на каждой странице (не запоминает выбор)
- Используются современные CSS-эффекты (backdrop-filter, box-shadow)
- Полностью адаптивный дизайн для всех устройств
- Соответствует GDPR и требованиям законодательства РФ
