Кастомные прошивки для Steam Deck (Windows, Batocera)

Экспертный мегагайд: пошаговая установка, сравнение, лайфхаки, FAQ, таблицы, иллюстрация.
Steam Deck — не только SteamOS: Windows и Batocera превращают консоль в универсальный геймерский комбайн. Как их установить, что выбрать и какие подводные камни? Подробный мегагайд — ниже.

Введение: зачем кастомные прошивки на Steam Deck?

Steam Deck из коробки работает на SteamOS — кастомной Linux-системе от Valve, идеально интегрированной с библиотекой Steam и оптимизированной для портативного гейминга. Но возможности устройства этим не ограничиваются: установка Windows открывает доступ к играм с античитом, Xbox Game Pass, лаунчерам и классическим приложениям, а Batocera превращает Deck в ретро-консоль с поддержкой сотен эмуляторов и удобным интерфейсом[5][6].

Выбор прошивки зависит от ваших задач: SteamOS — для максимального удобства и автономности, Windows — для универсальности и совместимости, Batocera — для ретро-гейминга и эмуляции[5][6].

Пошаговая инструкция: установка Windows на Steam Deck

  1. Скачайте ISO Windows 10/11 и Rufus на ПК.
  2. Вставьте microSD или SSD в ПК, запустите Rufus.
    • Выберите ISO, режим Windows To Go, отметьте все опции[3].
  3. Скачайте драйверы Steam Deck с сайта Valve, распакуйте их на microSD/SSD.
  4. Вставьте microSD/SSD в Steam Deck, зажмите Volume Down + Power для входа в Boot Manager.
  5. Выберите microSD/SSD, следуйте инструкции по установке Windows.
    • На этапе выбора Wi-Fi пропустите шаг — драйверы установите позже.
  6. После загрузки Windows установите драйверы вручную (Wi-Fi, Bluetooth, APU, аудио, SD-картридер)[3].
  7. Перезагрузите Deck, настройте Steam, лаунчеры, геймпад и энергосбережение.

Совет: Для максимального удобства используйте dual-boot — SteamOS на внутреннем SSD, Windows на microSD[3][5].

Пошаговая инструкция: установка Batocera на Steam Deck

  1. Скачайте образ Batocera для Steam Deck с официального сайта.
  2. Запишите образ на microSD с помощью Balena Etcher (можно прямо из SteamOS Desktop Mode)[4].
  3. Вставьте microSD в Steam Deck, выключите консоль.
  4. Зажмите Volume Down + Power для входа в Boot Manager.
  5. Выберите microSD с Batocera — система загрузится автоматически[4][6].
  6. Настройте BIOS и ROM-ы, подключите геймпад, выберите темы и эмуляторы.

Batocera работает полностью с microSD и не трогает внутренний SSD или SteamOS — идеальный вариант для ретро-гейминга без риска![4][6]

Лайфхаки и советы по кастомным прошивкам на Steam Deck

🔹 Практические советы

Для Batocera и Windows используйте отдельные microSD — не трогайте внутренний SSD, чтобы не потерять SteamOS[4][6].

В Batocera можно полностью кастомизировать интерфейс, темы, списки платформ и эмуляторов[4][6].

Для Windows обязательно ставьте все драйверы с сайта Valve — иначе не будет работать Wi-Fi, звук, Bluetooth и SD-картридер[3].

В SteamOS лучше всего работает эмуляция через EmuDeck — для большинства пользователей это оптимальный вариант[2].

Batocera позволяет полностью изолировать ретро-игры от основной системы — удобно для путешествий и семейного использования[4][6].

FAQ — часто задаваемые вопросы о кастомных прошивках Steam Deck

  • Можно ли полностью заменить SteamOS на Windows?
    Да, но рекомендуется dual-boot или запуск с microSD, чтобы не потерять удобство SteamOS[3][5].
  • Batocera удаляет SteamOS?
    Нет, она работает с microSD и не трогает внутренний диск — просто вытащите карту для возврата к SteamOS[4][6].
  • Какая прошивка лучше для эмуляции?
    Для ретро — Batocera, для современных игр с античитом — Windows, для универсальности — SteamOS с EmuDeck[2][4][5][6].
  • Можно ли запускать Windows и Batocera с одной карты?
    Нет, лучше использовать отдельные microSD для каждой системы[4][6].
  • Что делать, если не работает звук/интернет в Windows?
    Установить все драйверы с сайта Valve вручную после установки Windows[3].

Пошаговая инструкция: возврат к SteamOS после кастомных прошивок

  1. Выключите Steam Deck полностью.
  2. Извлеките microSD с Windows или Batocera.
  3. Включите Deck — автоматически загрузится SteamOS.
  4. Если SteamOS не загружается, восстановите систему через официальный образ с сайта Valve.

Итог: какую кастомную прошивку выбрать для Steam Deck?

Steam Deck — уникальная платформа, где можно легко переключаться между SteamOS, Windows и Batocera. Для большинства пользователей оптимален dual-boot или запуск альтернативных систем с microSD: так вы получаете максимум возможностей без риска потерять родную ОС. SteamOS — для комфорта и автономности, Windows — для универсальности, Batocera — для ретро-гейминга и эмуляции. Следуйте пошаговым инструкциям, используйте только официальные драйверы и быстрые карты памяти — и ваш Steam Deck станет универсальным игровым центром!
