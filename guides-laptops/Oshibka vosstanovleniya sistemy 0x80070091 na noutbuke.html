Ошибка восстановления системы 0x80070091 на ноутбуке

Ошибка 0x80070091 при восстановлении системы указывает на проблемы с каталогом. В этом подробном руководстве мы разберем причины этой ошибки и эффективные способы ее исправления для ноутбуков.

Что означает ошибка 0x80070091

Код 0x80070091 связан с каталогом, который не является пустым.

Техническое объяснение

  • Код ошибки: 0x80070091
  • Полное название: The directory is not empty
  • Причина: Каталог не пустой при попытке удаления
  • Часто связано: С точками восстановления или системными файлами

Влияние на систему

  • Не удается восстановить систему
  • Проблемы с точками восстановления
  • Нестабильная работа
  • Потеря возможности отката

Распространенные причины

Почему возникает ошибка 0x80070091 при восстановлении системы.

Проблемы с точками восстановления

  • Поврежденные точки: Точки восстановления повреждены
  • Недостаточно места: Мало места для восстановления
  • Конфликты файлов: Файлы блокируют восстановление
  • Антивирус: Блокировка системных процессов

Системные проблемы

  • Поврежденный реестр: Ошибки в реестре
  • Файлы в каталоге: Файлы мешают удалению
  • Разрешения: Недостаточно прав доступа
  • Диск поврежден: Ошибки на системном диске

Другие причины

  • Обновления Windows: После установки обновлений
  • Вредоносное ПО: Вирусы повредили точки восстановления
  • Драйверы: Конфликты с драйверами
  • BIOS настройки: Неправильные параметры

Диагностика ошибки

Как определить причину ошибки 0x80070091.

Проверка точек восстановления

Восстановление системы

  1. Откройте "Параметры" > "Обновление и безопасность"
  2. Выберите "Восстановление"
  3. Нажмите "Вернуть компьютер в исходное состояние"
  4. Посмотрите доступные точки восстановления
  5. Попробуйте восстановить - посмотрите на ошибку

Командная строка

  1. Откройте командную строку от имени администратора
  2. Введите: vssadmin list shadows
  3. Посмотрите список теневых копий
  4. Введите: vssadmin delete shadows /all
  5. Попробуйте восстановить систему

Проверка диска

  1. Откройте командную строку от имени администратора
  2. Введите: chkdsk C: /f /r
  3. Дождитесь завершения проверки
  4. Посмотрите отчет об ошибках

Решение проблемы

Методы исправления ошибки 0x80070091.

Метод 1: Удаление теневых копий

Очистка точек восстановления

  1. Откройте командную строку от имени администратора
  2. Введите: vssadmin delete shadows /all
  3. Подтвердите удаление
  4. Создайте новую точку восстановления
  5. Попробуйте восстановить систему

Метод 2: Отключение защиты системы

Временное отключение

  1. Откройте "Параметры" > "Обновление и безопасность"
  2. Выберите "Восстановление"
  3. Нажмите "Настроить"
  4. Отключите защиту системы
  5. Включите обратно после восстановления

Метод 3: Исправление реестра

Очистка поврежденных записей

  1. Создайте точку восстановления
  2. Откройте редактор реестра (regedit)
  3. Перейдите к HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SPP\Clients
  4. Удалите ключи с поврежденными данными
  5. Перезагрузите систему

Метод 4: Чистая загрузка

Отключение сторонних служб

  1. Откройте msconfig
  2. На вкладке "Службы" снимите все галочки
  3. На вкладке "Автозагрузка" отключите все
  4. Перезагрузите и попробуйте восстановить
  5. Включите службы по одной

Метод 5: Восстановление через установщик

Использование установочной флешки

  1. Загрузитесь с установочной флешки Windows
  2. Выберите "Восстановление системы"
  3. Выберите "Восстановление системы"
  4. Следуйте инструкциям

Специфические случаи

Решения для распространенных сценариев.

  • Отключите защиту системы
  • Запустите sfc /scannow

  • Отключите антивирус временно

  • Включите антивирус обратно

Ошибка на корпоративных системах

  • Свяжитесь с администратором
  • Проверьте групповые политики
  • Обновите сертификаты
  • Проверьте сетевые настройки

Профилактика

Как избежать ошибки 0x80070091.

Регулярное обслуживание

  • Точки восстановления: Регулярно создавайте точки
  • Чистка: Очищайте диск от мусора
  • Антивирус: Защита от вредоносного ПО
  • Обновления: Устанавливайте обновления своевременно

Безопасность восстановления

  • Создавайте точки восстановления перед изменениями
  • Регулярно проверяйте их работоспособность
  • Не удаляйте точки восстановления без необходимости
  • Следите за свободным местом

Заключение

Ошибка 0x80070091 при восстановлении системы обычно связана с поврежденными точками восстановления или конфликтами файлов. В большинстве случаев ее можно решить удалением теневых копий или отключением защиты системы. Начните с проверки точек восстановления, затем удалите теневые копии. Если самостоятельные попытки не увенчались успехом, обратитесь к специалистам или поддержке Microsoft.
