Проверить поддержку Windows 11 на ноутбуке — пошаговая инструкция для 2025 года

Разбираем, как за 5 минут понять, потянет ли конкретный ноутбук Windows 11: официальный PC Health Check, ручная проверка CPU, TPM 2.0, Secure Boot, ОЗУ и диска, а также типичные причины отказа.
Microsoft рекомендует использовать приложение PC Health Check для быстрой проверки совместимости, а при необходимости смотреть детали вручную: модель процессора, наличие TPM 2.0, Secure Boot, объём ОЗУ и диска.

Кратко: что именно должно совпасть

В 2025 году ноутбук считается совместимым с Windows 11, если у него: поддерживаемый 64?битный процессор (из официального списка), минимум 4 ГБ ОЗУ, 64 ГБ диска, UEFI?прошивка с включённым Secure Boot и активный TPM 2.0, плюс графика с DirectX 12/WDDM 2.0. Для веток 24H2+ CPU также должен поддерживать инструкции SSE4.2 и POPCNT, иначе установка/загрузка невозможна.

Таблица: что проверяем для поддержки Windows 11

Параметр Что требуется для Windows 11 Чем проверить
CPU 64?битный, 2+ ядра, в списке поддерживаемых; для 24H2 — с SSE4.2+POPCNT. «Параметры» > «Система» > «О системе» или wmic cpu get name, затем свериться со списком CPU Microsoft.
ОЗУ Минимум 4 ГБ, лучше 8 ГБ+. «Диспетчер задач» > «Производительность» или «О системе» в Windows.
Накопитель 64 ГБ+ свободного места (внутренний диск). Проводник > «Этот компьютер» (объём и свободное место).
TPM TPM 2.0, активен. Windows+R > tpm.msc или «Параметры» > «Обновление и безопасность» > «Безопасность устройства».
Secure Boot / UEFI Режим UEFI, Secure Boot включён. msinfo32 (пункты BIOS Mode и Secure Boot State) и настройки UEFI.

Способ 1. Быстрая проверка через PC Health Check

Официальное приложение PC Health Check (Проверка работоспособности ПК) автоматически оценивает ноутбук по всем требованиям и выдаёт простой вердикт «соответствует/не соответствует» с расшифровкой причин. Этот способ рекомендован Microsoft и производителями как основной.

  1. Скачать PC Health Check по ссылке https://aka.ms/GetPCHealthCheckApp или с официальной страницы Windows 11 («Check for compatibility»).
  2. Установить и запустить приложение, найти блок «Introducing Windows 11» и нажать «Check now» / «Проверить сейчас».
  3. Посмотреть итог: если появилось «This PC meets Windows 11 requirements», ноутбук поддерживается; если нет — нажать «See all results», чтобы увидеть, какой именно пункт не проходит.

Способ 2. Ручная проверка CPU и списков Microsoft

Microsoft ведёт отдельные списки поддерживаемых процессоров Intel и AMD для Windows 11; проверка модели CPU с ними гарантирует понимание, «проходной» у вас ноутбук или нет. Windows Central и другие гайды описывают два пути: через «О системе» и через командную строку.

  1. Открыть «Параметры» > «Система» > «О системе» и посмотреть строку «Процессор» и «Тип системы» (64?разрядная ОС, x64?процессор).
  2. Либо выполнить в командной строке: wmic cpu get name, чтобы увидеть точное название CPU.
  3. Свериться с официальными списками поддерживаемых процессоров (страницы Microsoft по Intel/AMD CPU для Windows 11).
  4. Если CPU отсутствует в списках или слишком старый (до примерно Intel 8?го поколения / Ryzen 2000), ноутбук формально не поддерживается.

Способ 3. Проверка TPM 2.0 и Secure Boot

TPM 2.0 и Secure Boot — ключевые требования безопасности для Windows 11; большинство ноутбуков последних лет имеют эти функции, но они могут быть выключены. Официальные инструкции предлагают два варианта проверки TPM: через tpm.msc и через «Параметры».

  • TPM. Нажать Windows+R > ввести tpm.msc > в открывшемся окне посмотреть «Specification version» (должно быть 2.0).
  • Secure Boot. Ввести в поиск msinfo32 > в «Сведения о системе» проверить поля «Режим BIOS» (должно быть UEFI) и «Состояние безопасной загрузки» (Вкл.).
  • Если TPM 2.0/ Secure Boot есть, но выключены, Microsoft рекомендует зайти в UEFI/BIOS и включить fTPM/TPM 2.0 и Secure Boot, затем снова запустить PC Health Check.

Типичные причины, почему ноутбук «не проходит» Windows 11

Причина в PC Health Check Что это значит
TPM 2.0 isn’t available Нет поддерживаемого TPM или он отключён; для многих ноутбуков решается включением fTPM/TPM 2.0 в BIOS.
Secure Boot isn’t enabled Система загружается в Legacy?режиме или Secure Boot отключён; нужно переключиться на UEFI и включить Secure Boot.
Processor isn’t supported CPU отсутствует в списке поддерживаемых; официальное обновление до Windows 11 недоступно.
Needs more RAM or storage ОЗУ меньше 4 ГБ или недостаточно места; решается апгрейдом памяти/очисткой или заменой диска.

Лайфхаки: как повысить шансы на совместимость

?? Что можно донастроить перед проверкой

Перед запуском PC Health Check зайдите в BIOS: включите UEFI?режим загрузки, Secure Boot и fTPM/TPM 2.0 — многие ноутбуки формально поддерживают Windows 11, но эти опции изначально выключены.

Если PC Health Check ругается только на ОЗУ/диск, проведите апгрейд (8 ГБ ОЗУ и SSD 256 ГБ+), затем повторите проверку — это одновременно подтянет и реальную скорость работы.

Если упираетесь только в «несовместимый процессор», а ноутбук критичен, имеет смысл оставить на нём Windows 10 и ограничить роль (узкие задачи/офлайн), а Windows 11 использовать уже на более новом железе.

?? Итог: как быстро понять, поддерживает ли ноутбук Windows 11

Проверка поддержки Windows 11 сводится к двум шагам: запуск PC Health Check с расшифровкой результатов и, при необходимости, ручная сверка модели CPU со списком Microsoft, а также проверка/включение TPM 2.0 и Secure Boot через tpm.msc, msinfo32 и UEFI. Если все ключевые требования — поддерживаемый 64?битный процессор, 4+ ГБ ОЗУ, 64+ ГБ диска, UEFI+Secure Boot и TPM 2.0 — выполняются, ноутбук официально готов к установке Windows 11 и получению обновлений в 2025 году.
